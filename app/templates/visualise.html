What's Happening?

Thresholding is a fundamental image processing technique in computer vision. It converts a grayscale image into a binary image based on a threshold value. Pixels above the threshold become white (1), and those below become black (0). This is useful for separating objects from the background, especially in segmentation tasks.

Try changing the threshold slider and uploading different images to see how the algorithm behaves. More tools like edge detection and blurring will be added soon to help understand how computers "see" images.
